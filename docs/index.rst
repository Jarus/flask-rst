Flask-RST
=========

.. module:: flaskrst

Flask-RST is a website generator to create static or dynamic websites builded from base of restructuredText files. Flask-RST is mainly inspired by `Armin Ronacher`_ `rstblog`_ but builded on top of his powerful `Flask`_ package.

.. _armin ronacher: https://github.com/mitsuhiko/
.. _rstblog: https://github.com/mitsuhiko/rstblog
.. _flask: http://flask.pocoo.org
